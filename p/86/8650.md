`**Document.hidden**`（只读属性）返回布尔值，表示页面是（true）否（false）隐藏。


## 语法<a name="语法"></a>

```
var string = document.hidden
```

## 例子<a name="例子"></a>

```
document.addEventListener("visibilitychange", function() {
  console.log( document.hidden );
  // Modify behavior...
});
```
