The**`HTMLDListElement`**interface provides special properties (beyond those of the regular[`HTMLElement`](%12142 "The HTMLElement interface represents any HTML element. Some elements directly implement this interface, others implement it via an interface that inherits it.")interface it also has available to it by inheritance) for manipulating definition list elements.
