`**clear()**`方法用来清空一个`Set`对象中的所有元素。


## 语法<a name="Syntax"></a>

```
mySet.clear();

```

### 返回值<a name="返回值"></a>


[`undefined`]23495 "undefined是全局对象的一个属性。也就是说，它是全局作用域的一个变量。undefined的最初值就是原始数据类型undefined。").


## 示例<a name="Examples"></a>

```
var mySet = new Set();
mySet.add(1);
mySet.add("foo");

mySet.size;       // 2
mySet.has("foo"); // true

mySet.clear();

mySet.size;       // 0
mySet.has("bar")  // false
```
