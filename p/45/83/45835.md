`**size**`可访问属性返回[`Map`]40774 "此页面仍未被本地化, 期待您的翻译!")对象的元素数量.


## 描述<a name="描述"></a>


size 属性的值是一个整数，表示 Map 对象有多少个键值对。size 是只读属性，它对应的 set 方法是 undefined，即不能改变它的值。


## 示例<a name="示例"></a>

```
var myMap = new Map();
myMap.set("a", "alpha");
myMap.set("b", "beta");
myMap.set("g", "gamma");

myMap.size // 3
```
