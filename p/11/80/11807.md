### 概述<a name="概述"></a>


**onmousedown**属性用来获取或设置当前元素的`mousedown`事件的事件处理函数.


### 语法<a name="语法"></a>

```
element.onmousedown= event handling code
```

### 备注<a name="备注"></a>


在当前元素上点击鼠标按键时会触发`mousedown事件.`


### 规范<a name="规范"></a>


不属于任何公开的规范.
