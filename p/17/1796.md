## 内置过渡动画<a name="nei-zhi-guo-du-dong-hua"></a>


Element 内应用在部分组件的过渡动画，你也可以直接使用。在使用之前请阅读[transition 组件文档](%1792 "")。


### fade 淡入淡出<a name="fade-dan-ru-dan-chu"></a>


提供`el-fade-in-linear`和`el-fade-in`两种效果。




===vue
<template><div>


</div></template>


<script>
module.exports =  {
    data: () => ({
      show: true
    })
  }
</script>

<style>
  .transition-box {
    margin-bottom: 10px;
    width: 200px;
    height: 100px;
    border-radius: 4px;
    background-color: #409EFF;
    text-align: center;
    color: #fff;
    padding: 40px 20px;
    box-sizing: border-box;
    margin-right: 20px;
  }
</style>
===

### 按需引入<a name="an-xu-yin-ru"></a>

```
// fade/zoom 等
import 'element-ui/lib/theme-chalk/base.css';
// collapse 展开折叠
import CollapseTransition from 'element-ui/lib/transitions/collapse-transition';
import Vue from 'vue'

Vue.component(CollapseTransition.name, CollapseTransition)

```
