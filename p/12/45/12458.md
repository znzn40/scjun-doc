The**`HTMLFormElement.target`**property represents the target of the action (i.e., the frame to render its output in).


## Syntax<a name="Syntax"></a>

```
string = form.target
form.target = string
```

## Example<a name="Example"></a>

```
myForm.target = document.frames[1].name;
```
