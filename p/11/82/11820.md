### 概述<a name="概述"></a>


**onmouseup**属性用来获取或设置当前元素的`mouseup`事件的事件处理函数.


### 语法<a name="语法"></a>

```
element.onMouseUp = event handling code
```

### 备注<a name="备注"></a>


当用户在当前元素上放开鼠标某个按键时会触发`mouseup`事件.


### 规范<a name="规范"></a>


不属于任何公开的规范.
