### 概述<a name="Summary"></a>


`applets`按顺序返回当前文档中所有的`applet`对象.


### 语法<a name="Syntax"></a>

```
nodeList = document.applets
```

### 例子<a name="Example"></a>

```
my_java_app = document.applets[1]; //获取当前页面的第二个applet对象.
```
