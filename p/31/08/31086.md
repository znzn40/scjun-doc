**Deprecated**<br></br>This feature has been removed from the Web standards. Though some browsers may still support it, it is in the process of being dropped. Avoid using it and update existing code if possible; see the[compatibility table](%35438 "")at the bottom of this page to guide your decision. Be aware that this feature may cease to work at any time.





The**`device-aspect-ratio`**[CSS](%28421 "")[media feature](%34551 "")can be used to test the width-to-height aspect ratio of an output device.


## Syntax<a name="Syntax"></a>


The`device-aspect-ratio`feature is specified as a[`<ratio>`](%28333 "The <ratio> CSS data type, used for describing aspect ratios in media queries, denotes the proportion between two unitless values."). It is a range feature, meaning that you can also use the prefixed**`min-device-aspect-ratio`**and`**max-device-aspect-ratio**`variants to query minimum and maximum values, respectively.


## Example<a name="Example"></a>

```
article {
  padding: 1rem;
}

@media screen and (min-device-aspect-ratio: 16/9) {
  article {
    padding: 1rem 5vw;
  }
}
```
