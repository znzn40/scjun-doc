当在window对象上触发close事件时的事件处理器 (在Firefox 2 或 Safari中不可用)


## 语法<a name="语法"></a>

```
window.onclose = funcRef;

```

### 参数<a name="参数"></a>

* `funcRef`是一个function的引用。

## 示例<a name="示例"></a>

```
window.onclose = resetThatServerThing;
```
