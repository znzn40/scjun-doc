### 概述<a name="概述"></a>


**onkeydown**属性用来获取或设置当前元素的`keydown`事件的事件处理函数.


### 语法<a name="语法"></a>

```
element.onkeydown = event handling code
```

### 备注<a name="备注"></a>


当用户按下键盘上的按键时会触发`keydown`事件.


### 规范<a name="规范"></a>


不属于任何公开的规范.
