### 概述<a name="Summary"></a>


`document.close()`方法用来关闭向当前文档的数据写入, 想要打开使用[document.open()]26116 "en/DOM/document.open").


### 语法<a name="Syntax"></a>

```
document.close();
```

### 例子<a name="Example"></a>

```
// 打开一个文档,向其中写入数据
// 写入文档内容
// 关闭文档
document.open();
document.write("<p>文档内容</p>");
document.close();
```
