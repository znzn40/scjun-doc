**Size**属性将会返回[`Set`]40561 "Set 对象允许你存储任何类型的唯一值，无论是原始值或者是对象引用。")对象中元素的个数。


## 说明<a name="说明"></a>


Size的值表示并且该值不能够被更改。


## 例子<a name="例子"></a>

### 使用<a name="使用"></a>

```
var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add("some text")

mySet.size; // 3
```
