`**columnNumber**`属性包含引发此错误的文件行中的列号。


## 例子<a name="例子"></a>

### 使用`columnNumber`<a name="使用_columnNumber"></a>

```
var e = new Error('Could not parse input');
throw e;
console.log(e.columnNumber) // 0
```
