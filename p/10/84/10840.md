## Summary<a name="Summary"></a>


Returns the size of a file in bytes.


## Syntax<a name="Syntax"></a>

```
var size = instanceOfFile.size
```

## Value<a name="Value"></a>


A number


## Specification<a name="Specification"></a>


Not part of any specification.
