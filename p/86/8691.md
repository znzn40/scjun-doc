## 概述<a name="Summary"></a>


返回跳转或打开到当前页面的那个页面的[URI](%25780 "")。


## 语法<a name="Syntax"></a>

```
var string = document.referrer;
```

## 备注<a name="Notes"></a>


如果用户直接打开了这个页面（不是通过页面跳转，而是通过地址栏或者书签等打开的），则该属性为空字符串。由于该属性只是返回一个字符串，所以不能够通过该属性引用页面的 DOM。
