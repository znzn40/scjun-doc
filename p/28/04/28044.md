## 语法<a name="Syntax"></a>

```
/* <length> values */
margin-right: 20px;        /* An absolute length */
margin-right: 1em;         /* A length relative to the text size */
margin-right: 5%;          /* A margin relative to the nearest block container's width */

/* Keyword values */
margin-right: auto;

/* Global values */
margin-right: inherit;
margin-right: initial;
margin-right: unset;
```
