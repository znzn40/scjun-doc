CSS 的 skewY() 函数被用来构建某一元素在垂直方向上进行特定角度偏转后形成的剪切映射。这一映射是通过将其纵坐标添加一个增量实现的，这一增量和特定角度以及距离原点的距离成正比。距离原点越远的点，添加的增量就越大。


## 语法<a name="语法"></a>

```
skewY(a)

```

## 实例<a name="实例"></a>

### HTML<a name="HTML"></a>

```
<p>foo</p>
<p class="transformed">bar</p>
```

### CSS<a name="CSS"></a>

```
p { 
  width: 50px;
  height: 50px;
  background-color: teal;
}

.transformed {
  transform: skewY(40deg);
  background-color: blue;
}
```
