## 概述<a name="Summary"></a>


`**Math.log10()**`函数返回一个数字以 10 为底的对数.


## 语法<a name="Syntax"></a>

```
Math.log10(x)
```

### 参数<a name="Parameters"></a>
<dl><dt id=''>`x`</dt><dd>任意数字.</dd></dl>
## 描述<a name="Description"></a>


如果传入的参数小于 0, 则返回 NaN.


## 示例<a name="Examples"></a>

```
Math.log10(10)   // 1
Math.log10(100)  // 2
Math.log10("100")// 2
Math.log10(1)    // 0
Math.log10(0)    // -Infinity
Math.log10(-2)   // NaN
Math.log10("foo")// NaN
```
