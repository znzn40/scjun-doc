
## 示例<a name="示例"></a>

### HTML<a name="HTML"></a>

```
<p>foo</p>
<p class="transformed">bar</p>
<p>foo</p>
```

### CSS<a name="CSS"></a>

```
p { 
  width: 50px;
  height: 50px;
  background-color: teal;
}

.transformed {
  transform: translateY(10px);
  background-color: blue;
}
```
