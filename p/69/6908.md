## 概述<a name="Summary"></a>


返回一个指向当前 window 对象的引用。


## 语法<a name="Syntax"></a>

```
objRef = window.self
```

## 示例<a name="Example"></a>

```
if (window.parent.frames[0] != window.self) {
    // 当前对象不是frames列表中的第一个时
 }
```

## 备注<a name="Notes"></a>


<b>window.self</b>几乎总是用于上面示例那样的比较，用来判断当前 window 是不是父 frameset 中的第一个 frame。


## 规范<a name="Specification"></a>
